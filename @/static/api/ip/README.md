# 说明

返回一些ip信息。

| 项目             | 说明    |
| ---------------- | ------- |
| 接口路径         | /api/ip |
| 接口请求数据类型 | 无      |
| 接口返回数据类型 | JSON    |

注意:当以浏览器访问该接口时,默认会跳转到说明文档，如需查看接口结果需要在查询参数中添加`forceapi=true`。

# 接口返回数据

数据类型为JSON.

| 字段 | 说明     |
| ---- | -------- |
| ip   | 客户端ip |

# 测试

```bash
# 以部署到syscall.online为例，采用curl测试
curl syscall.online/api/ip/
```
