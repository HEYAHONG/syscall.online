# 说明

返回一些日出日落信息。

| 项目             | 说明                  |
| ---------------- | --------------------- |
| 接口路径         | /api/sunriseandsunset |
| 接口请求数据类型 | 查询参数              |
| 接口返回数据类型 | JSON                  |

注意:当以浏览器通过GET方法访问该接口时,默认会跳转到说明文档，如需查看接口结果需要在查询参数中添加`forceapi=true`。

# 接口请求数据

数据类型为查询参数。

| 查询字段 | 说明                 | 备注                         |
| -------- | -------------------- | ---------------------------- |
| lat      | 纬度，双精度浮点数   | 默认为成都的纬度（30.5728）  |
| lon      | 精度，双精度浮点数   | 默认为成都的经度（104.0688） |
| year     | 年，整型             | 默认为服务器本地时间         |
| month    | 月，整型             | 默认为服务器本地时间         |
| day      | 日，整型             | 默认为服务器本地时间         |
| timezone | 时区，整型           | 默认为UTC+8(值为8)           |
| zenith   | 天顶角，双精度浮点数 | 默认为标准天顶角             |

# 接口返回数据

数据类型为JSON.

| 字段     | 说明                 | 备注                                                         |
| -------- | -------------------- | ------------------------------------------------------------ |
| lat      | 纬度，双精度浮点数   | 同请求时                                                     |
| lon      | 精度，双精度浮点数   | 同请求时                                                     |
| year     | 年，整型             | 同请求时                                                     |
| month    | 月，整型             | 同请求时                                                     |
| day      | 日，整型             | 同请求时                                                     |
| timezone | 时区，整型           | 同请求时                                                     |
| zenith   | 天顶角，双精度浮点数 | 同请求时                                                     |
| sunrise  | 日出时间，JSON对象   | 包含h（小时，整型）、m（分钟，整型）、s(秒，整型）三个子对象 |
| sunset   | 日落时间，JSON对象   | 包含h（小时，整型）、m（分钟，整型）、s(秒，整型）三个子对象 |

# 测试

```bash
# 以部署到syscall.online为例，采用curl测试
curl syscall.online/api/sunriseandsunset/
```
